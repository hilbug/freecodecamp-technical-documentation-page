First Section

This document will walk you through creating a basic fan page for a TV show using Bootstrap 5 components and classes and minimal custom CSS. The fan page will consist of:

  • Navigation bar
  • Hero Section
  • Character bios
  • Extra reading
  • YouTube video

Bootstrap 5 is a CSS framework which allows you to create mobile responsive websites. It has many built-in components with lots of classes to choose from. It is built upon Flexbox Utilities for easy positioning. A bootstrap page consists of rows and columns. The page will have one container made up of rows and each row contains 12 columns.

To get started, two CDN links should be added to your html. First, add the CDN link for Bootstrap in the head tag BEFORE your local CSS stylesheet: 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-eOJMYsd53ii+scO/bJGFsiCZc+5NDVN2yr8+0RDqr0Ql0h+rP48ckxlpbzKgwra6" crossorigin="anonymous">

Next, add the CDN link for Bootstrap's JavaScript bundle at the end of the body: 

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/js/bootstrap.bundle.min.js" integrity="sha384-JEW9xMcG8R+pH31jmWH6WWP0WintQrMb4s7ZOdauHnUtxwoG2vI5DkLtS3qm9Ekf" crossorigin="anonymous"></script>

Finally, let's create the container and the first row and column in the <body> for the navigation bar and hero section: 

                            
                                <div class="container-fluid">
                                    <div class="row">
                                        <div class="col-12 p-0">
                                            Navigation and Hero section will go inside here
                                        </div>
                                    </div>
                                </div>

A few notes about what we have added:

  • The class .container-fluid so that it spans the whole width of the screen.
  • The class .p-0 on the column removes the padding around the column itself so the navbar is the whole width of the screen.

To learn more about Bootstrap, visit the Getting Started section of their website.

Second Section

A nice thing about Bootstrap are its reusable components. We are going to include a simple nav with links to the character bios, additional reading, and the YouTube video. For this project, I am just going to use a .nav component. Inside the col-12, we will add the following:

                                
                                    <nav class="nav justify-content-center bg-info bg-gradient" role="navigation">
                                        <a class="nav-link" href="#">Characters</a>
                                        <a class="nav-link" href="#">Articles</a>
                                        <a class="nav-link" href="#">Video</a>
                                      </nav>

Next, we will create the hero section. First, let's add the background image via CSS. We will give the hero section the class .hero. The height will be set to 50vh (50% of the viewport height). You can use any image you like. An easy way to get an image from the internet is to right-click and select Copy Image Address: 

                            
                                .hero {
                                    background-image: url("https://miro.medium.com/max/2560/1*cRmipwCGD5drJv6tQNNwBQ.jpeg");
                                    background-position: center;
                                    background-size: auto;
                                    background-repeat: no-repeat;
                                    height: 50vh;
                                }

Next, let's add the hero section to the HTML and add header text with the TV show's name. This will be added below the nav:

                            
                                <div class="hero p-5 bg-light d-flex align-items-end justify-content-center">
                                    <h1 class="rounded bg-info text-center text-light p-2">Dawson's Creek</h1>
                                </div>

A few notes about what we have added:

  • The classes .d-flex, .align-items-end, .justify-content-center allow us to position the h1 in the center at the bottom of the div because Bootstrap uses FlexBox utilities.
  • The classes .rounded, .text-center, .text-light, and .bg-info allow us to transform the h1 using Bootstrap classes.

To learn more about:

  • nav components, visit Navs & tabs section.
  • flex utilities, visit Flex section.

Third Section

We are now going to add a section for characters. Each character will have its own card using another Bootstrap component. Below the first row, let's add a new div which will contain the character cards. Each card will have a photo, the character's name, and a brief bio. The cards will be centered on the page and will leverage row column classes to preserve the alignment. Here is an example of the div with one character card. Each card image was given a height of 329px in CSS:

                            
                                <!-- Character Cards -->
                                <div id="characters" class="row bg-info bg-gradient d-flex justify-content-center">
                                    <h2 class="text-center p-2 bg-white rounded w-100 mt-3 text-info">Character Bios</h2>
                                    <div class="col-9 p-3">
                                        <!-- CARD ROWS -->
                                        <div class="row row-cols-1 row-cols-md-2 g-4">
                                            <!-- Joey Potter -->
                                            <div class="col">
                                                <div class="card shadow-lg rounded">
                                                    <img src="https://static.onecms.io/wp-content/uploads/sites/13/2015/09/02/Screen-Shot-2015-09-02-at-12.12.48-PM.jpg"
                                                        class="card-img-top" alt="joey potter smiling at camera">
                                                    <div class="card-body">
                                                        <h5 class="card-title text-center">Josephine Potter</h5>
                                                        <p class="card-text">Josephine Potter, known as Joey, is a childhood friend of Dawson
                                                            Leery. She lives with her sister, sister's boyfriend, and nephew. Her mother died of
                                                            cancer when she was young. Her father is in prison for trafficking drugs. She is one
                                                            of the smartest students at Capeside High and hopes to attend Wentworth upon
                                                            graduation.</p>
                                                    </div>
                                                </div>
                                            </div>
                                        </div>
                                    </div>
                                </div>

A few notes about what we have added:

  • The card row classes row row-cols-1 row-cols-md-2 g-4 allow us to leverage Bootstrap's grid to neatly position the cards.
  • The classes .shadow-lg rounded allow us to style and add dimension to the cards.

To learn more about:

  • Bootstrap cards, visit the Card section.
  • Grid cards, visit Card > Grid cards section.
